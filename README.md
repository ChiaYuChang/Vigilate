# Go WebSockets 練習

- [Working with WebSockets in Go (Golang)](https://www.udemy.com/course/working-with-websockets-in-go/)
- [Course code respiratory](https://github.com/tsawler/ws-udemy)

# 目標

- 以 Go 建立一個即時監測網路服務的網路應用程式 (Web application)
- 程式需具備以下功能
    - 使用者登入/登出
    - 加入/刪除監測服務
        - 定時檢測服務是否還在線
        - 紀錄每次檢測結果
        - 檢測完成後即時更新結果
